## 🌊 What is AURA?

</div>

> **AURA** is a beautifully designed personal journaling web app built with pure HTML, CSS, and JavaScript — zero frameworks, zero build tools, zero dependencies.  
> It lives in your browser, your data stays yours, and it feels like nothing else you've used before.

## 🚀 Quick Start

</div>

> ⚠️ **Important:** Always run on a local server — never open as `file://`. Browser security blocks `localStorage` and `crypto.subtle` on raw file paths.

<br/>

### Option A — VS Code Live Server ⭐ Recommended

```bash
# 1. Open the aura/ folder in VS Code
# 2. Install "Live Server" extension (ritwickdey.LiveServer)
# 3. Right-click auth.html → "Open with Live Server"
# ✅ Opens at http://127.0.0.1:5500/aura/auth.html
```

### Option B — Python

```bash
cd aura
python -m http.server 8080
# ✅ Open: http://localhost:8080/auth.html
```

### Option C — Node.js

```bash
cd aura
npx serve .
# ✅ Open the URL shown in your terminal
```

<div align="center">

## 🔥 Enable Google & Facebook Login

</div>

Email/password works out of the box. To unlock social login, follow these steps:

<br/>

**Step 1 — Create Firebase Project**

```
1. Go to → https://console.firebase.google.com
2. Click "Add project" → Name it "aura-journal"
3. Disable Google Analytics (optional) → Create project
```

**Step 2 — Enable Authentication Providers**

```
Firebase Console
  └── Build
       └── Authentication
            └── Sign-in method
                 ├── ✅ Email/Password  → Enable → Save
                 ├── ✅ Google          → Enable → Add support email → Save
                 └── ✅ Facebook        → Enable → Add App ID & Secret → Save
```

**Step 3 — Get Your Config**

```
Firebase Console
  └── ⚙️ Project Settings
       └── General
            └── Your apps
                 └── </> Web
                      └── Register app → Copy firebaseConfig
```

**Step 4 — Paste Into AURA**

Open `auth.html` and find this block (~line 180):

```javascript
// ─── PASTE YOUR FIREBASE CONFIG BELOW ──────────────────
const FIREBASE_CONFIG = {
  apiKey:            "YOUR_API_KEY",         // ← replace
  authDomain:        "YOUR_PROJECT.firebaseapp.com",
  projectId:         "YOUR_PROJECT_ID",
  storageBucket:     "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId:             "YOUR_APP_ID"
};
```

**Step 5 — Add Authorized Domains**

```
Firebase Console → Authentication → Settings → Authorized domains
Add:
  ✅ localhost
  ✅ 127.0.0.1
  ✅ YOUR_USERNAME.github.io   ← for GitHub Pages
```

<div align="center">

## 🧠 How It Works

</div>

```
┌─────────────────────────────────────────────────────────┐
│                    USER OPENS BROWSER                    │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│                      auth.html                          │
│                                                         │
│  • Check localStorage for existing session             │
│  • If session exists → redirect to index.html          │
│  • Email/Pass → SHA-256 hash → localStorage accounts   │
│  • Google/FB  → Firebase OAuth popup → session         │
│  • On success → save user → navigate to app            │
└────────────────────────┬────────────────────────────────┘
                         │  ✅ Authenticated
                         ▼
┌─────────────────────────────────────────────────────────┐
│                      index.html                         │
│                                                         │
│  • Guard: no session → back to auth.html               │
│  • Load user from localStorage → populate UI           │
│  • Load entries from localStorage (aura_entries_v4)    │
│  • Render home tab → real stats → real cards           │
│                                                         │
│  ┌──────────┬───────────┬──────────┬──────────────┐   │
│  │  Home    │ Calendar  │  Moods   │   Search     │   │
│  │  Stats   │  Month    │  Bars    │  Live filter │   │
│  │  Cards   │  Dots     │  Log     │  Full-text   │   │
│  └──────────┴───────────┴──────────┴──────────────┘   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │           Editor Modal (contenteditable)        │   │
│  │  Rich text · Mood picker · Tags · Word count    │   │
│  │  Save → localStorage → re-render all views     │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```
